# 宠物医院后台管理系统 - 管理后台前端

## 技术栈

- Vue 3 + TypeScript
- Vite 7
- Element Plus
- Pinia（状态管理）
- Vue Router 4
- Axios

## 页面模块

- 登录（支持公司端/门店端切换）
- 首页仪表盘
- 组织架构：公司信息、门店管理
- 员工管理
- 客户管理
- 宠物管理
- 系统管理：角色管理、菜单管理

## 本地运行

```bash
npm install
npm run dev
```

开发服务器默认运行在 http://localhost:5173，API 请求代理到 http://localhost:8080。
